# DDR-Clone

#### Building

`$ git clone --recurse-submodules https://github.com/aup9qmb/DDR-Clone.git`\
`$ make`

The game binary, once built with `$ make`, can be found in `./bin/`.

Run with `$ ./bin/game` out of the root directory of the repository.
